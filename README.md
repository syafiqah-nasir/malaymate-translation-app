# MalayMate - Bilingual Malay-English Translation App

## 📱 Overview

MalayMate is a bilingual translation mobile application designed to bridge the communication gap between international tourists and local residents in Malaysia. The app provides real-time translation services between English and Malay, including support for 14 Malaysian state dialects.

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Text Translation** | Translate written text between English and Malay |
| **Camera Translation** | OCR-based real-time translation of signs, menus, and documents |
| **Voice Translation** | Speech-to-text and text-to-speech for live conversations |
| **Offline Phrasebook** | Essential phrases in 14 dialects (no internet required) |

### Supported Dialects for Offline Phrasebook
Standard Malay, Selangor, Johor, Kedah, Kelantan, Melaka, Negeri Sembilan, Pahang, Penang, Perak, Perlis, Sabah, Sarawak, Terengganu

## 🛠️ Tech Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Flask (Python)
- **NLP Library**: Malaya
- **OCR**: PaddleOCR
- **Database**: SQLite
- **Version Control**: GitHub

## 🎯 Project Objectives

- [x] Provide accurate translation between standard Malay and English
- [x] Implement real-time camera translation using OCR
- [x] Implement real-time voice translation for conversations
- [x] Provide offline phrasebook with local dialects
